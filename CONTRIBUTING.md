# Contributing

Thanks for considering a contribution!

## Quick Setup

```
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```

## Running Tests

```
pytest
```

## Linting

```
ruff check .
```

## Pull Requests

- Keep changes focused and small.
- Add or update tests when possible.
- Update documentation if behavior changes.
